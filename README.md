starbound_scripts
=================

Simple Starbound scripts

Make sure the scripts are in the same directory as linux32/ linux64/ assets/ etc...

	starbound/
		|
		|--- assets/
		|
		|--- linux32/
		|
		|--- mods/
		|
		|--- package/
		|
		|--- panel.sh
		|
		|--- public/
		|
		|--- Starbound.app/
		|
		|--- start.sh
		|
		|--- win32/
		|
	There are also two logfiles from my scripts:
		|
		|--- mem.log.txt
		|
		|--- restart.log.txt
